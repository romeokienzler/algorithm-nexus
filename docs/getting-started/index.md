# Getting Started

Welcome to Algorithm Nexus! This guide will help you get started with installing
and using the Nexus CLI tool for managing AI model packages.

## What is Algorithm Nexus?

Algorithm Nexus is a library that robustly packages a diverse set of AI models
and frameworks within a unified Python environment, enabling seamless deployment
and management of multiple models with different dependencies.

## Quick Start

1. **[Install Algorithm Nexus](installation.md)** - Get the CLI tool and
   dependencies set up
2. **[Understand Nexus Packages](../design/nexus_package.md)** - Learn about the
   package structure
3. **[Start Contributing](../contributing/index.md)** - Add your own models and
   packages
